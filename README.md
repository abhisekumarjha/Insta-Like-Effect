# Insta Like Effect

This project creates a visually appealing "like" effect inspired by social media platforms, using HTML, CSS, and JavaScript. The effect is triggered by a double-click on an image, showcasing an animated heart icon.

## Features

- Display an image with a title.
- Smooth animation of a heart icon on double-click.
- Responsive design that adapts to different screen sizes.

## Technologies Used

- HTML5
- CSS3
- JavaScript
